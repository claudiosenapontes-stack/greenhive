# Greenhive Electric

A professional institutional website for Greenhive Electric, a leading electrical services company providing residential and commercial electrical solutions.

## Features

- **Modern Design**: Clean, professional aesthetic with green/blue color scheme
- **Responsive Layout**: Fully responsive design for desktop, tablet, and mobile
- **Sections Include**:
  - Hero with call-to-action
  - About the company
  - Services (Residential, Commercial, Repairs, Installations, Energy Solutions, Inspections)
  - Why Choose Us
  - Contact form with company info
- **Interactive Elements**: Smooth scrolling, form handling, scroll animations

## Technologies

- HTML5
- CSS3 (Custom properties, Grid, Flexbox)
- Vanilla JavaScript
- Google Fonts (Inter)

Visit: https://claudiosenapontes-stack.github.io/greenhive/

To run locally:
